## A = B+C-D

```ASSEMBLY
	AREA	RESET, CODE
		ENTRY
		
		MOV R1,#4 ;RANDOM VALUES
		MOV R2,#5
		MOV R3,#2
		
		ADD R0,R1,R2
		SUB R0,R0,R3
STOP B STOP
		END
```


## A = 2* A + B

```ASSEMBLY
	AREA	RESET, CODE
		ENTRY
		
		MOV R0,#4
		MOV R1,#5
		
		LSL R0,#1 ;MULTIPLYING BY 2 = LEFT SHIFTING BY 1
		ADD R0,R0,R1
STOP B STOP
		END
```
